# Control Maps — Customer Invoice Export API

Coarse, evidence-level traceability for this example feature. These are
traceability aids, not certification claims.

| Framework | Relevant Area | Evidence |
|---|---|---|
| CSA CCM v4.1 | IAM (identity & access) | authorize_export tests, denial logs |
| CSA CCM v4.1 | Logging & Monitoring | audit log of export decisions |
| NIST SSDF (SP 800-218) | PW (Produce Well-Secured Software) | negative/security tests, threat model |
