# Documentation Checklist Addendum

- [ ] Data classification is complete for docs, diagrams, screenshots, examples, logs, and evidence.
- [ ] Tagging discipline is applied to docs, diagrams, evidence, risks, and exceptions.
- [ ] Documentation impact is recorded.
- [ ] Public docs are updated where behavior changes.
- [ ] Contributor/operator docs are updated where workflows change.
- [ ] Diagrams are source-controlled.
- [ ] Docs-site build passes.
