# Control Maps

These maps provide traceability from SicarioSpec evidence to external control
and audit concerns. They are not certification claims and do not copy full
control text from third-party frameworks.

Current starter maps:

- `ccm-v4.1-sicario.json`: CSA Cloud Controls Matrix v4.1 domain-level map.
- `sox-404-itgc-sicario.json`: SOX Section 404 / ICFR ITGC evidence map.
- `ssdf-800-218-sicario.json`: NIST SSDF (SP 800-218) practice-group evidence map.
- `ai-rmf-sicario.json`: NIST AI RMF (AI 100-1) Govern/Map/Measure/Manage evidence map.

Each map is a starter that links SicarioSpec evidence to framework outcomes at a
coarse level (domain, practice group, or function). They are not control-by-control
crosswalks and not certification claims.

Use the official framework source when performing a real audit, attestation, or
customer-facing control assertion.
