# Extensions

The first extension is `sicario-guard`.

Commands:

- `/sicario.init`
- `/sicario.assess`
- `/sicario.threatmodel`
- `/sicario.controls`
- `/sicario.evidence`
- `/sicario.verify`
- `/sicario.review`
- `/sicario.apply-findings`

The extension is designed for Spec Kit command discovery. The deterministic
local implementation lives in the `sicario` CLI.
