# System Context

SicarioSpec is installed into target projects as local governance assets:

- `.specify/presets/*`
- `.specify/extensions/sicario-guard`
- docs and diagram scaffolds
- CCM/SOX control map starters
- risk and exception registers
- CI workflow templates

The deterministic verifier reads repository files and writes evidence under
`generated/sicario/`.

Diagram source: `docs/diagrams/system-context.mmd`
