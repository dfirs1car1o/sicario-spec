# Support

Use the right channel so maintainers can respond without mixing public support
with private security concerns.

| Need | Channel |
|---|---|
| Bug report | GitHub issue form: Bug report |
| Feature idea | GitHub issue form: Feature request |
| Control mapping request | GitHub issue form: Control map or governance request |
| Usage question | GitHub Discussions |
| Private vulnerability | GitHub Security Advisory flow |

## Response Expectations

This is an early public project. Maintainers will prioritize:

1. Security-sensitive reports.
2. Install, release, and packaging breakages.
3. Verification false negatives or false positives.
4. Control map correctness.
5. New profiles and examples.

SicarioSpec does not provide audit, legal, compliance certification, or
professional services through public support channels.
