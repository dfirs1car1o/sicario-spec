# `/sicario.init`

Initialize SicarioSpec files in the current repository.

Actions:

1. Verify `.specify/` exists.
2. Verify SicarioSpec presets are installed.
3. Verify `docs/security`, `docs/compliance`, `docs/diagrams`, and `docs-site`
   exist.
4. Run `sicario verify`.
5. Report next steps.

Never write outside the repository.
